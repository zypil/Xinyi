# 🌙 XINYI - VESPERA Discord Bot

**Auto-leveling | Leaderboard | Admin System | Nitro Rewards**

## Setup

1. Upload to GitHub
2. Import to Replit
3. Add `.env` secrets in Replit
4. Run `npm install`
5. Click **Run**

## Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/profile` | View profile card | Everyone |
| `/leaderboard` | View leaderboard | Everyone |
| `/rank` | Check rank & progress | Everyone |
| `/setleaderboardinfo` | Set leaderboard channel | Admin |
| `/setcommandchannel` | Set admin command channel | Admin |
| `/setwelcome` | Set welcome channel | Admin |
| `/setautorole` | Set auto-role | Admin |
| `/givexp` | Give XP | Admin |
| `/removexp` | Remove XP | Admin |
| `/resetuser` | Reset user data | Admin |
| `/nitroreward` | Award Nitro top 3 | Owner |
| `/serverstats` | Server statistics | Admin |
| `/config` | View configuration | Admin |
| `/help` | Show all commands | Everyone |

## Features

- ✅ Auto-register new members
- ✅ XP gain from chatting (1 min cooldown)
- ✅ 7 Rank tiers with custom colors
- ✅ Streak system
- ✅ Beautiful profile cards (Canvas)
- ✅ Auto-updating leaderboard image
- ✅ Admin-only private command channel
- ✅ Top 3 leaderboard = Free Nitro 3 Days
- ✅ Firebase Realtime Database
